# Compatibilidad

Esta sección reúne la documentación orientada a los desarrolladores de ElephanTalk. Aquí se describen aspectos relacionados con el mantenimiento del proyecto, las buenas prácticas de desarrollo y la información técnica necesaria para garantizar la continuidad y evolución del sistema.

El objetivo es proporcionar una referencia para los miembros del equipo y futuros colaboradores, facilitando el mantenimiento, la incorporación de nuevas funcionalidades y la gestión de los componentes utilizados por el proyecto.
